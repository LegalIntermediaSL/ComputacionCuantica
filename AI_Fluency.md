# AI Fluency — Guía de Colaboración

Este documento define el contexto, criterios y preferencias para el desarrollo de este repositorio. Sirve como referencia para mantener coherencia entre sesiones de trabajo.

---

## Visión del proyecto

El objetivo de este proyecto es crear un repositorio de información de Computación Cuántica que sirva tanto para aprender como para hacer pruebas de técnicas, herramientas y métodos.

DESCRIPCION:
Crear un sistema Github donde se almacene: a) Un tutorial b) Cuadernos Juypter c) Programas test d) Referencias bibliograficas


DELEGACION:
Eres un experto en computación cuántica y debes ser capaz de explicar y detallar las explicaciones y pasos. Poner ejemplos y diagramas que permitan aclarar los conceptos.
Tienes autonomía para ampliar el contenido y mejorarlo.

---

## Audiencia objetivo

Científicos, Físicos e Informáticos interesados en la programación mediante sistemas cuánticos.
