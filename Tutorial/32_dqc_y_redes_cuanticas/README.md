# Módulo 32 — Computación Cuántica Distribuida y Redes Cuánticas

**Nivel:** muy avanzado · **Prerrequisitos:** módulos 09, 14, 29, 31

Este módulo estudia cómo conectar procesadores cuánticos independientes mediante
canales cuánticos para construir redes cuánticas y ordenadores cuánticos distribuidos.

## Artículos

1. [01_entanglement_swapping_y_repetidores.md](01_entanglement_swapping_y_repetidores.md)
   — Teletransportación de largo alcance, purificación de pares EPR, repetidores cuánticos
2. [02_dqc_y_protocolos_de_computo.md](02_dqc_y_protocolos_de_computo.md)
   — Computación cuántica distribuida: delegated QC, blind QC, protocolos non-local gates
3. [03_qkd_y_criptografia_cuantica.md](03_qkd_y_criptografia_cuantica.md)
   — BB84, E91, QBER, Twin-Field QKD, post-quantum cryptography vs. QKD

## Contexto

La computación cuántica a gran escala requerirá conectar módulos de procesadores
mediante redes fotónicas. Los experimentos actuales de entanglement swapping
(2023: distancias > 1000 km vía satélite) muestran la viabilidad de las redes cuánticas.
