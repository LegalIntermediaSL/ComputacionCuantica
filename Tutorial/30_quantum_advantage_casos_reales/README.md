# Módulo 30 — Quantum Advantage en la práctica

**Nivel:** avanzado · **Prerrequisitos:** módulos 11, 12, 28, 29

Este módulo analiza los casos de uso donde la computación cuántica puede ofrecer
ventaja real frente a los mejores algoritmos clásicos conocidos, con ROI cuantificado
y horizonte temporal realista.

## Artículos

1. [01_quimica_femoco_y_simulacion.md](01_quimica_femoco_y_simulacion.md) — Simulación del catalizador FeMoco, ventaja cuántica en química, estimaciones de recursos
2. [02_optimizacion_financiera_qubo.md](02_optimizacion_financiera_qubo.md) — QUBO a escala, portafolios, arbitraje estadístico y límites prácticos del QAOA actual
3. [03_logistica_y_combinatoria.md](03_logistica_y_combinatoria.md) — TSP cuántico, scheduling, optimización de rutas y comparativa con heurísticas clásicas

## Objetivos de aprendizaje

- Distinguir entre supremacía cuántica (demostración de capacidad) y ventaja cuántica (utilidad práctica).
- Estimar los recursos cuánticos necesarios para superar algoritmos clásicos en problemas de química, finanzas y logística.
- Identificar el horizonte temporal realista para cada caso de uso.
