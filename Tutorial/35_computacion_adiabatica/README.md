# Módulo 35 — Computación Cuántica Analógica y Adiabática

**Nivel:** muy avanzado · **Prerrequisitos:** módulos 11, 18, 20, 29

La computación cuántica analógica explota la dinámica continua de sistemas cuánticos
para resolver problemas de optimización, en contraste con el modelo gate-based discreto.

## Contexto

La computación adiabática cuántica (AQC) es polinomialmente equivalente al modelo
gate-based en teoría, pero presenta ventajas prácticas en problemas de optimización
combinatoria cuando el gap espectral se mantiene suficientemente grande.
