# Módulo 34 — Sistemas Cuánticos Abiertos

**Nivel:** muy avanzado · **Prerrequisitos:** módulos 16, 17, 19, 29

Los sistemas cuánticos reales nunca están perfectamente aislados. Este módulo
estudia la dinámica de sistemas que interactúan con el entorno (baño térmico),
y cómo caracterizar esa interacción experimentalmente.

## Artículos

1. [01_ecuacion_lindblad_y_dinamica.md](01_ecuacion_lindblad_y_dinamica.md)
   — Ecuación de Lindblad: derivación, operadores de salto, canales de depolarización, termalización
2. [02_tomografia_de_procesos.md](02_tomografia_de_procesos.md)
   — Tomografía de procesos cuánticos (QPT): χ-matrix, PTM, SPAM errors, GST

## Contexto

La decoherencia es el obstáculo central para la computación cuántica fault-tolerant.
Entender la ecuación de Lindblad y la tomografía de procesos es esencial para
diseñar estrategias de mitigación de errores y QEC eficientes.
