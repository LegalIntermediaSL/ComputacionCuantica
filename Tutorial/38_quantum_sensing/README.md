# Módulo 38 — Quantum Sensing y Metrología

**Nivel:** muy avanzado · **Prerrequisitos:** módulos 15, 17, 23

La metrología cuántica explota la superposición y el entrelazamiento para
medir magnitudes físicas con precisión más allá del límite estándar cuántico
(SQL). Este módulo cubre la física fundamental y las aplicaciones emergentes.

## Artículos

1. [01_sensores_cuanticos.md](01_sensores_cuanticos.md)
   — Límite de Heisenberg, estados GHZ para metrología, relojes atómicos,
   magnetómetros NV-center, Fisher información cuántica
2. [02_aplicaciones_sensing.md](02_aplicaciones_sensing.md)
   — Gravímetros cuánticos, navegación inercial, radar cuántico, interferometría
   atómica, LIGO y límites cuánticos de medición

## Contexto

| Límite | Precisión | Recurso |
|--------|-----------|---------|
| Clásico (shot noise) | 1/√N | N medidas independientes |
| SQL (Standard Quantum Limit) | 1/√N | Coherencia cuántica |
| Límite de Heisenberg | 1/N | Entrelazamiento máximo (GHZ) |

La ventaja cuántica en metrología es **demostrable hoy** con hardware NISQ,
a diferencia de la computación cuántica de propósito general.
