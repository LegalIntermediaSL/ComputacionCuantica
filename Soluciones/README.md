# Soluciones

Este directorio reune soluciones seleccionadas de ejercicios del proyecto.

## Criterio

No se pretende resolver inmediatamente todos los ejercicios. La idea es:

- dejar trabajo real al lector;
- ofrecer algunas soluciones modelo;
- mostrar estilo de razonamiento y no solo respuestas finales.
