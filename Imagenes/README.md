# Imagenes

Este directorio esta reservado para diagramas, figuras y material visual del tutorial.

## Estructura

- `diagramas/`
- `figuras/`
- `referencias/`

## Uso recomendado

- usar `diagramas/` para Mermaid exportado o esquemas propios;
- usar `figuras/` para imagenes finales que se enlacen desde el tutorial;
- usar `referencias/` para material visual de apoyo que no forme parte todavia del cuerpo principal.
