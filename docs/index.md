## Visualizador interactivo

Lanza la aplicación Streamlit localmente:

```bash
pip install -r requirements.txt
streamlit run visualizador/app.py
```

8 páginas interactivas: Esfera de Bloch, Algoritmos, Canales y Ruido, Hardware Dashboard,
VQE/QAOA, Corrección de Errores, Compositor de Circuitos, Tomografía de Estados.
