# Contributing

## Objetivo

Este proyecto crece como un tutorial modular sobre computacion cuantica y Qiskit. Si se añaden materiales nuevos, conviene mantener coherencia entre teoria, practica y documentacion.

## Principios

- Priorizar claridad pedagogica antes que densidad innecesaria.
- Mantener la notacion consistente entre modulos.
- Enlazar, cuando sea posible, teoria con cuadernos asociados.
- Evitar duplicar contenido si puede integrarse mejor en un modulo existente.

## Nuevos articulos

Cuando se cree un articulo nuevo, conviene:

1. ubicarlo en el modulo correcto;
2. incluir objetivo y foco del documento;
3. usar formulas en LaTeX cuando sean utiles;
4. añadir ideas clave o ejercicios sugeridos.

## Nuevos cuadernos

Cuando se cree un notebook nuevo, conviene:

1. indicar objetivo al inicio;
2. dejar claro si trabaja con estado ideal, cuentas o hardware/simulacion;
3. mantener el codigo enfocado y legible;
4. relacionarlo con uno o varios articulos del `Tutorial/`.

## Documentacion del proyecto

Si se hacen cambios relevantes, actualizar tambien:

- `README.md`
- `bitacora.md`
- `changelog.md`

segun corresponda.
