# Laboratorios

Los laboratorios son notebooks mas largos que los ejemplos y mas abiertos que los problemas resueltos.

## Objetivo

Servir como espacios de exploracion guiada donde se combinan:

- teoria minima;
- implementacion en Qiskit;
- interpretacion de resultados;
- preguntas para extender el experimento.

## Cobertura inicial

- Grover con dos qubits;
- teleportacion guiada;
- transpilacion y comparacion;
- Bernstein-Vazirani guiado;
- QFT con tres qubits;
- phase estimation guiada;
- VQE guiado;
- QAOA guiado.
- barrido de energia tipo VQE;
- paisaje de coste tipo QAOA.
